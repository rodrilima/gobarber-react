## Getting Started

### Prerequisites

To run this project, you'll only need to have the npm or yarn installed.

### Installing

Cloning the Repository

```
$ git clone https://github.com/rodrilima/gobarber-react.git

$ cd gobarber-react

$ yarn
```

### Running

```
$ yarn start
```
